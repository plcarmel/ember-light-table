Defined in addon/mixins/table-header.js:77

CSS classes to be applied to an <i class="lt-sort-icon"></i> tag that is inserted into the column's <th> element when the column is sortable but not yet sorted.

For instance, if you have installed ember-font-awesome or include the font-awesome assets manually (e.g. via a CDN), you can set iconSortable to 'fa fa-sort', which would yield this markup: <i class="lt-sort-icon fa fa-sort"></i>
